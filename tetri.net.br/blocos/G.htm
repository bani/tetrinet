- Block Gravity -
G

O G, além de ser um ótimo bloco de defesa, pode ser utilizado para adicionar linhas no seu adversário. Para isto, vá empilhando os blocos de forma a deixar uma coluna com quantidade menor. Quando um bloco reto estiver caindo, utilize o G para fazer um Tetris, adicionando assim 4 linhas no seu adversário se o Classic Mode estiver habilitado. Em seguida, também aproveite os blocos em L para fazer linhas duplas ou triplas. Além de adicionar as linhas, você receberá os blocos especiais que estavam nelas multiplicados pelo número de linha feitos de cada vez.
